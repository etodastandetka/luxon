# Инструкция по обновлению на сервере

## Обновление кода и сервисов

```bash
# 1. Перейти в директорию проекта
cd /var/www/luxon

# 2. Обновить код из Git
git pull origin main

# 3. Перезапустить админку (Next.js)
cd admin_nextjs
pm2 restart luxon-admin-nextjs

# 4. Перезапустить бота (для получения сообщений в чат)
pm2 restart luxon-bot

# 5. Проверить статус всех процессов
pm2 status

# 6. Проверить логи бота (должны появиться сообщения о сохранении в чат)
pm2 logs luxon-bot --lines 50

# 7. Проверить логи админки
pm2 logs luxon-admin-nextjs --lines 50
```

## Проверка работы

1. **Проверка чата:**
   - Откройте админ-панель
   - Перейдите в чат с пользователем
   - Попросите пользователя отправить сообщение боту
   - Сообщение должно появиться в админ-чате через 3-5 секунд

2. **Проверка кнопок подтверждения:**
   - Откройте заявку на пополнение со статусом "Ожидает"
   - Нажмите "Подтвердить"
   - После пополнения баланса кнопки должны исчезнуть
   - Статус должен обновиться автоматически

## Если что-то не работает

1. **Бот не получает сообщения:**
   ```bash
   # Проверить, что бот запущен
   pm2 status luxon-bot
   
   # Проверить логи на ошибки
   pm2 logs luxon-bot --err --lines 100
   
   # Проверить, что httpx установлен
   cd /var/www/luxon/django_admin
   source venv/bin/activate
   pip list | grep httpx
   ```

2. **Админка не обновляется:**
   ```bash
   # Пересобрать Next.js проект
   cd /var/www/luxon/admin_nextjs
   npm run build
   pm2 restart luxon-admin-nextjs
   ```

3. **Сообщения не сохраняются в чат:**
   - Проверить логи бота на наличие ошибок API
   - Проверить, что API endpoint `/api/users/[userId]/chat/ingest` доступен
   - Проверить логи админки на ошибки сохранения в БД
