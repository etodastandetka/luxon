# 🔧 Исправление ошибки "Chat not found"

## Проблема
Бот не может найти канал или не имеет прав на публикацию.

## ✅ Решение

### Шаг 1: Найдите вашего бота
Ваш бот: **@instdownload123_bot**

### Шаг 2: Добавьте бота в канал как администратора

1. Откройте канал `@luxon_news_test` в Telegram
2. Нажмите на **название канала** (вверху)
3. Выберите **"Администраторы"** или **"Administrators"**
4. Нажмите **"Добавить администратора"** или **"Add Administrator"**
5. Найдите бота **@instdownload123_bot** или введите его username
6. **ВАЖНО**: Включите права **"Отправка сообщений"** (Post Messages)
7. Нажмите **"Готово"** или **"Done"**

### Шаг 3: Проверьте ID канала

Текущий ID канала: `-1003444253148`

Если канал публичный, можно использовать username: `@luxon_news_test`

Чтобы изменить ID канала, отредактируйте файл `config.py`:
```python
TELEGRAM_CHANNEL_ID = "@luxon_news_test"  # Для публичного канала
# или
TELEGRAM_CHANNEL_ID = "-1003444253148"  # Для приватного канала
```

### Шаг 4: Перезапустите бота

```bash
python bot.py
```

Бот автоматически проверит доступ к каналу и отправит тестовое сообщение.

## 🔍 Проверка

После запуска бота вы должны увидеть:
```
✅ Канал найден: название_канала
✅ Тестовое сообщение отправлено успешно
✅ Доступ к каналу подтвержден
```

Если видите ошибку, следуйте инструкциям выше.
