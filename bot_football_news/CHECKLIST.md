## 📝 Дополнительные настройки (опционально):

### AI форматирование:
Если хотите использовать AI для форматирования:
1. Зарегистрируйтесь на [Hugging Face](https://huggingface.co/)
2. Получите API токен
3. В `config.py` измените:
   ```python
   AI_TEXT_SERVICE = "huggingface"
   HUGGINGFACE_API_KEY = "ваш_токен"
   ```

### Генерация изображений:
Если хотите генерировать изображения:
1. Получите API ключ для Hugging Face или другого сервиса
2. В `config.py` измените:
   ```python
   AI_IMAGE_SERVICE = "huggingface"
   ENABLE_IMAGE_GENERATION = True
   HUGGINGFACE_API_KEY = "ваш_токен"
   ```
