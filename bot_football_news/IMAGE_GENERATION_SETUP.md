# 🖼️ Настройка генерации изображений

## ✅ Что уже сделано

1. **Генерация изображений включена** по умолчанию (`ENABLE_IMAGE_GENERATION = true`)
2. **Убрана ссылка "Подробнее"** из постов
3. **Улучшено форматирование текста** - теперь новости выглядят как полноценные статьи
4. **Изображения генерируются с описанием** новости для лучшего качества

## 🔧 Настройка сервиса генерации изображений

В файле `config.py` можно выбрать один из сервисов:

### Вариант 1: Hugging Face (рекомендуется для начала)

1. Зарегистрируйтесь на [Hugging Face](https://huggingface.co/)
2. Создайте API токен в [настройках](https://huggingface.co/settings/tokens)
3. Добавьте в `config.py`:
   ```python
   AI_IMAGE_SERVICE = "huggingface"
   HUGGINGFACE_API_KEY = "ваш_токен"
   ```

### Вариант 2: FusionBrain (Kandinsky от Сбера)

1. Зарегистрируйтесь на [FusionBrain](https://fusionbrain.ai/)
2. Получите API ключи
3. Добавьте в `config.py`:
   ```python
   AI_IMAGE_SERVICE = "fusionbrain"
   FUSIONBRAIN_API_KEY = "ваш_api_key"
   FUSIONBRAIN_SECRET_KEY = "ваш_secret_key"
   ```

### Вариант 3: Replicate

1. Зарегистрируйтесь на [Replicate](https://replicate.com/)
2. Получите API токен
3. Добавьте в `config.py`:
   ```python
   AI_IMAGE_SERVICE = "replicate"
   REPLICATE_API_TOKEN = "ваш_токен"
   ```

### Вариант 4: Отключить генерацию

Если не хотите использовать генерацию изображений:
```python
ENABLE_IMAGE_GENERATION = False
```

## 📝 Формат новостей

Теперь новости публикуются в формате статьи:

- **Заголовок** (жирный, с эмодзи)
- **Вступление** (в цитате, как начало статьи)
- **Детали матча** (структурированная информация)
- **Анализ и прогноз** (развернутый текст)
- **Хештеги**
- **Изображение** (генерируется автоматически)

## 🚀 Запуск

После настройки просто запустите бота:

```bash
python bot.py
```

Бот будет автоматически:
1. Получать новости из AllSportsAPI
2. Форматировать их как статьи
3. Генерировать изображения с описанием
4. Публиковать в канал

## 💡 Примечание

Если API ключ для генерации изображений не настроен, бот будет работать без изображений, но с улучшенным текстовым форматом.
