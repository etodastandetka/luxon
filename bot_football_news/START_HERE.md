## ⚠️ Важно перед запуском

**Убедитесь, что:**
1. Бот добавлен как **администратор** в канал `@luxon_news_test`
2. У бота есть права на **публикацию сообщений** в канале

**Как добавить бота в канал:**
1. Откройте канал в Telegram
2. Нажмите на название канала → "Администраторы"
3. Нажмите "Добавить администратора"
4. Найдите вашего бота по токену или username
5. Дайте права на "Отправка сообщений"

## 🚀 Запуск бота

```bash
python bot.py
```

Бот:
1. Подключится к Telegram
2. Выполнит тестовую публикацию новости
3. Запустит планировщик для автоматической публикации

## 📅 Расписание публикаций

По умолчанию бот публикует новости:
- По времени: `09:00`, `15:00`, `21:00` (МСК)
- По интервалу: каждые 6 часов

Настройки можно изменить в файле `config.py`:
```python
PUBLISH_TIMES = ["09:00", "15:00", "21:00"]
PUBLISH_INTERVAL_HOURS = 6
```

## 🔍 Проверка работы

## 🛠️ Настройка AI (опционально)

Сейчас бот работает **без AI** (простое форматирование).

Чтобы включить AI форматирование:
1. Зарегистрируйтесь на [Hugging Face](https://huggingface.co/)
2. Создайте API токен
3. Добавьте в `.env`:
   ```
   AI_TEXT_SERVICE=huggingface
   HUGGINGFACE_API_KEY=ваш_токен
   ```

Для генерации изображений:
```
AI_IMAGE_SERVICE=huggingface
ENABLE_IMAGE_GENERATION=true
HUGGINGFACE_API_KEY=ваш_токен
```
