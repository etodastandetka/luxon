# 🆓 Бесплатные сервисы для генерации изображений

## ✅ Что добавлено

Добавлена поддержка **бесплатных** сервисов для генерации изображений:

### 1. **Craiyon** (рекомендуется) ⭐
- ✅ **Полностью бесплатный** - без регистрации и API ключей
- ✅ **Работает сразу** - не требует настройки
- ✅ **Без ограничений** - можно использовать сколько угодно
- ⚠️ Может быть медленнее (30-60 секунд на генерацию)

**Настройка:**
```python
# В config.py уже установлено по умолчанию
AI_IMAGE_SERVICE = "craiyon"
ENABLE_IMAGE_GENERATION = True
```

### 2. **Stability AI** (требует API ключ)
- ✅ Бесплатный tier доступен
- ✅ Высокое качество изображений
- ⚠️ Требует регистрации и API ключ

**Настройка:**
1. Зарегистрируйтесь на https://platform.stability.ai/
2. Получите бесплатный API ключ
3. В `config.py`:
   ```python
   AI_IMAGE_SERVICE = "stabilityai"
   STABILITYAI_API_KEY = "ваш_ключ"
   ```

### 3. **FusionBrain (Kandinsky)** (требует API ключи)
- ✅ Бесплатный от Сбера
- ✅ Хорошее качество
- ⚠️ Требует регистрации

**Настройка:**
1. Зарегистрируйтесь на https://fusionbrain.ai/
2. Получите API ключи
3. В `config.py`:
   ```python
   AI_IMAGE_SERVICE = "fusionbrain"
   FUSIONBRAIN_API_KEY = "ваш_key"
   FUSIONBRAIN_SECRET_KEY = "ваш_secret"
   ```

### 4. **Replicate** (бесплатный tier)
- ✅ Бесплатный tier с ограничениями
- ⚠️ Требует регистрации
