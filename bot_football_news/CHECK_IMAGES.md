## ❌ Если изображения не генерируются

### Проблема 1: Сервис установлен в "none"

**Решение**: В `config.py` измените:
```python
AI_IMAGE_SERVICE = "huggingface"  # вместо "none"
```

### Проблема 2: Генерация отключена

**Решение**: В `config.py` проверьте:
```python
ENABLE_IMAGE_GENERATION = True  # должно быть True
```

### Проблема 3: Ошибка API

**Решение**: Проверьте логи на ошибки:
- "Hugging Face API ключ не установлен" - проверьте токен
- "HTTP ошибка" - проверьте интернет-соединение
- "Модель загружается" - подождите, модель загрузится автоматически

### Проблема 4: Изображение не отправляется

**Решение**: Проверьте:
- Файл изображения создается в папке `images/`
- Бот имеет права на отправку фото в канал
- Размер изображения не превышает лимиты Telegram
