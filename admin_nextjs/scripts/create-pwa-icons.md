# Создание иконок для PWA

Для создания иконок PWA выполните следующие шаги:

1. Создайте иконку 512x512 пикселей с зеленым фоном и текстом "LUXON" или логотипом
2. Сохраните её как `icon-512.png` в папке `public/`
3. Создайте уменьшенную версию 192x192 пикселей и сохраните как `icon-192.png` в папке `public/`

Или используйте онлайн-генератор:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

После создания иконок удалите файлы `icon-192.png` и `icon-512.png` из `.gitignore` (если они там есть) и закоммитьте их.
