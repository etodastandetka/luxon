# Координаты для Mob-Cash API

## Проблема

Ошибка "cashbox out of work area" означает, что координаты не соответствуют рабочей области кассы.

## Как найти правильные координаты

### Способ 1: Из браузера (рекомендуется)

1. Откройте https://app.mob-cash.com в браузере
2. Войдите в систему
3. Откройте DevTools (F12) → Network
4. Найдите запрос `mobile.login` к `admin.mob-cash.com/api/`
5. Откройте вкладку **Payload**
6. Найдите параметр `location`:

```json
{
  "jsonrpc": "2.0",
  "id": 12,
  "method": "mobile.login",
  "params": {
    "location": {
      "lat": 34.6805775,    ← это правильная широта
      "lon": 33.0458273     ← это правильная долгота
    },
    "cashboxCode": 1001098,
    "userID": "1985621306577461248"
  }
}
```

**Скопируйте эти координаты** и используйте их в `.env`:

```env
MOBCASH_DEFAULT_LAT="34.6805775"
MOBCASH_DEFAULT_LON="33.0458273"
```

### Способ 2: Использовать координаты из мобильного приложения

Если у вас есть мобильное приложение mob-cash:
1. Откройте приложение
2. Войдите в систему
3. Координаты будут автоматически определяться по GPS
4. Эти координаты можно использовать

### Способ 3: Использовать координаты кассы

Координаты должны соответствовать реальному местоположению кассы. Если касса находится в определенном месте, используйте координаты этого места.

## Текущие координаты в коде

По умолчанию используются:
- Широта (lat): `34.6805775`
- Долгота (lon): `33.0458273`

Эти координаты соответствуют примеру из документации, но могут не подходить для вашей кассы.

## Решение

1. **Найдите правильные координаты из браузера** (способ 1 - самый надежный)
2. Добавьте их в `.env`:
   ```env
   MOBCASH_DEFAULT_LAT="ваша_широта"
   MOBCASH_DEFAULT_LON="ваша_долгота"
   ```
3. Перезапустите приложение

## Примечание

Если координаты все равно не работают, возможно:
- Касса не настроена на работу с удаленными координатами
- Нужно использовать координаты, которые были использованы при первоначальной настройке кассы
- Свяжитесь с поддержкой mob-cash для уточнения рабочей области кассы
