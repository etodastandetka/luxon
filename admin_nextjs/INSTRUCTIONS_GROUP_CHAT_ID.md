# Инструкция: Как получить ID Telegram группы

## Способ 1: Через бота @userinfobot

1. Добавьте бота [@userinfobot](https://t.me/userinfobot) в вашу группу
2. Отправьте любое сообщение в группу (или попросите бота отправить)
3. Ответьте на это сообщение командой `/start` боту @userinfobot
4. Бот пришлет информацию о группе, включая ID

## Способ 2: Через бота @getidsbot

1. Добавьте бота [@getidsbot](https://t.me/getidsbot) в вашу группу
2. Отправьте команду `/start` боту в группе
3. Бот покажет ID группы

## Способ 3: Через API Telegram

1. Добавьте вашего бота в группу и сделайте его администратором
2. Отправьте любое сообщение в группу
3. Используйте API метод `getUpdates`:
   ```bash
   curl "https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates"
   ```
4. Найдите в ответе поле `chat` с `id` - это и есть ID группы

## Способ 4: Через веб-версию Telegram

1. Откройте группу в веб-версии Telegram (web.telegram.org)
2. Посмотрите URL - он будет содержать ID группы
3. Или используйте консоль браузера (F12) и выполните:
   ```javascript
   window.Telegram.WebApp.initDataUnsafe.chat.id
   ```

## Важно!

- **ID частной группы** обычно начинается с `-100` (например: `-1001234567890`)
- **ID публичной группы/канала** может быть как положительным, так и отрицательным
- **ID супергруппы** всегда начинается с `-100`

## Настройка в проекте

После получения ID группы, добавьте его в файл `.env`:

```env
GROUP_CHAT_ID=-1001234567890
```

Или передайте при вызове функции `sendTelegramGroupMessage`:

```typescript
await sendTelegramGroupMessage(message, '-1001234567890')
```

## Проверка

После настройки, проверьте что сообщения приходят в группу. Если сообщения не приходят:

1. Убедитесь, что бот добавлен в группу
2. Убедитесь, что бот имеет права на отправку сообщений
3. Проверьте правильность ID группы (должен начинаться с `-100` для частных групп)
4. Проверьте логи на наличие ошибок
